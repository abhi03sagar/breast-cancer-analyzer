# Breast Cancer Analysis & Prediction

This project analyzes the **Breast Cancer Wisconsin (Diagnostic) Dataset** and builds a machine learning model to classify tumors as either **Malignant** or **Benign**.

## Project Overview

The script performs the following steps:
1.  **Data Loading**: Loads the dataset directly from `scikit-learn`.
2.  **Exploratory Data Analysis (EDA)**:
    *   Visualizes the distribution of diagnosis classes (Malignant vs. Benign).
    *   Generates a correlation heatmap for the first 10 features.
3.  **Preprocessing**: Splits the data into training/testing sets and scales features using `StandardScaler`.
4.  **Model Training**: Trains a Support Vector Machine (SVM) classifier with a linear kernel.
5.  **Evaluation**: Outputs model accuracy, a confusion matrix, and a detailed classification report.

## Requirements

*   Python 3.x
*   pandas
*   numpy
*   matplotlib
*   seaborn
*   scikit-learn

## Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

Run the main script to execute the analysis and training pipeline:

```bash
python main.py
```

## Results

Upon running the script, you will see:
*   Two pop-up windows with visualization plots.
*   Console output showing the dataset shape, model accuracy, confusion matrix, and classification metrics.
